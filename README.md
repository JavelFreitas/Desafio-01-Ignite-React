<h2 id="pre-requisitos">Pre-requisitos</h2>

* Node (v16^)
* Git
* Yarn

<h2 id="instalar-projeto">Como instalar o projeto</h2>
<h3>
Baixar Projeto 
</h3>

```
git clone https://github.com/JavelFreitas/Desafio-01-Ignite-React.git
```
<h3>
Instalar as Dependências 
</h3>

```
yarn
```
<h3>
Criar uma build 
</h3>

```
yarn build
```
<h3>
Rodar 
</h3>

```
yarn dev
```
<h3>
Testar (Opcional) 
</h3>

```
yarn test
```
